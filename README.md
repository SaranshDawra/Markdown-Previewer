## Setup
#### Clone this repository
```bash
git clone https://github.com/SaranshDawra/Markdown-Previewer.git
```
#### Move to desired directory
```bash
cd Markdown-Previewer
```
#### To install the dependencies
```bash
npm install
```
#### To run the app
```bash
npm start
```
